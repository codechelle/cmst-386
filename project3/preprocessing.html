Project 3 Preprocessing

Implementing Preprocessors

Preprocessors can be installed on the client-side and server-side. Via client-side, developers can use graphical user interfaces, like Scout App to convert their SASS files to CSS. Via server-side, developers can install globally or directly to a project using Node. Installation via server-side installs the preprocessor directly to the project and processes through module bundlers like Webpack. Webpack will compile the SASS to CSS once npm is ran to update the json package.

SASS

Nesting

Nesting is one of the best things about SASS. Instead of having to write a parent element over and over again to target its children elements and states, you can nest them instead and avoid having long lines of code to style a page. This is even greater for larger websites - think company and e-commerce.

Mixins

Mixins allow developers to group css properties that would be declared multiple times to style the HTML elements. It keeps the style documents clean and concise.

Operators

Operators help support all the amazing functionality that SASS provides. Sometimes we need to do the math to get precise sizing, but that's a time consumer. With the help of operators, developers can use the to calculate (CSS) values.
